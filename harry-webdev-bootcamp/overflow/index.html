Overflow: visible (default)

This is some long content that will overflow outside the box. It will not be cut or scrollable.

Overflow: hidden

This is some long content that will overflow but will be hidden. You cannot see extra content.

Overflow: scroll

This is some long content that will overflow and always show scrollbars even if not needed.

Overflow: auto

This is some long content that will overflow and show scrollbars only when needed.
